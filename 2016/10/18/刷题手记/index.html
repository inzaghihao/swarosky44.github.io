Bit Counting

描述

将一个数转化成二进制，统计其中有多少个 1

eg: 1234 => 10011010010 => 5

实现

可优化部分

这里推荐两个答案

是不是有点懵逼 … n >>= 1 && c += n & 1 这是啥呀 !

不要慌，我们先看看两个位运算符的 >> 和 &

>>: 有符号右运算符

eg:

&: 按位与运算符

所以上面的代码案例，我们可以解析为：

  1. 第一次数字 n 进来之后判断 n 是否为 0 ？

  2. 不为 0 则执行 c += n & 1, 如果 n 二进制数中最后一位不是 1 则 n & 1 等于 0，c 保持不变，相反 c += 1

  3. 代码执行完之后对 n 进行赋值操作 n 整体右移一位，则二进制最后一位右移。

  4. 持续循环，直到 n 为 0

其实这个案例实现的确十分巧妙，但是也相当难以阅读。并且会改变参数 n 的值，实际开发中其实一点都不推荐，不过能复习一下位运算符也是很不错的 😝

这是第二个推荐答案

这个看上去就比较便于阅读了，只要掌握了 toString(2) 的含义就能够轻易实现

toString() 是 Object.prototype 上的一个方法，本意是用于将调用者的值转换成字符串，常用于转字符串或者类型判断

但是! NumberObject.prototype.toString(radix) 上会接受一个参数

radix 表示基数，将数字转化成以基数为紧致的表示形式(默认十进制)。如果调用对象不是 Number ,则会抛出 TypeError 异常。
