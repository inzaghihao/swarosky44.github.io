第二次探索

经过了前面两次实现单例，我们总是得不到一个令人满意的方案，这是因为前两种方案都是传统面向对象语言的实现方式。而这里指的是类似于 Java 一类的语言，他们的对象必须来自于类。

但是 JS 是基于原型链的面向对象语言呀 … 我们又不需要先创建类再创建对象。

我们可以直接在全局环境下通过字面量的形式创建一个实例。

这个 a 对象是否就满足了单例模式的定义呢 ?

其实这也就满足了单例模式，因为 JS 毕竟是无类的语言，生搬硬套传统的单例模式意义也不大。
