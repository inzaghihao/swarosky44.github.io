Array对象

概述

Array 是 JS 的内置对象,同时 Array() 也是一个构造函数，可以用来生成新的数组对象。向构造函数传递不同的参数，Array 构造函数会做出不同的响应。

数组的每一项可以保存任何类型的数据类型，并且数组对象的长度是动态的，即数组的 length 属性可以根据数组中元素的增减实现变化。

在 JS 中 Array 的 length 属性不是只读属性，可以通过修改 length 属性实现对数组内元素的增减。

2.Array.isArray()

Array.isArray(obj) 是 es5 添加的新方法，目的用于判断某个对象是否是数组，而不用担心是哪个全局环境下创建的。

Array 转换方法

toString() 、 toLocalString() 、 valueOf()是所有对象都有的三个方

valueOf() 返回数组本身

toString() 返回一个用逗号拼接数组内元素的字符串

toLocalString() 大部分时候会返回和 toString() 相同的值。但实际上不同的是 toString() 是对每个元素进行 toString()，而toLocalString() 方法则是对每个元素进行 toLocalString() 方法

Array实例的方法

非数组需要使用 [].method.call 或者Array.prototype.method.call 才可以使用的方法：

1.push() 向数组末端添加一个元素，返回值为数组的长度，如果添加元素为一个对象，则实际添加的是对象的引用

2.pop() 移除数组末端最后一个元素，返回值为移除的元素，如果移除空数组，返回undefined

3.join() 以参数为分隔符，组成一个字符串为返回值

4.contact() 合并两个数组对象，返回一个新的数组对象

5.shift() 删除数组的第一个元素，并返回删除的元素

6.unshift()在数组的第一个位置添加元素，返回数组的长度

7.reverse()颠倒数组的元素的排序

8.slice([startIdx],[endIdx])截取数组，返回新数组对象为截取的数组

9.splice([startIdx],[length],[...arg])删除数组中的一部分，返回被删除的部分数组，第一个参数为删除的起始位置，第二个参数为删除数组的长度，第三个~第N个为在删除位置将添加的新元素。

12.sort([function])对数组元素进行排序，通过将元素调用 toString() 方法转换成字符串然后进行字符串对比。可以自定义排序，传入一个函数对象为参数

ES5新增方法

1.map(function(elem、index、arr){})传入一个参数为函数对象，遍历数组元素传入函数中，根据函数返回值返回一个新数组

2.forEach(function(ele, idx, arr){})类似map()但是没有返回值

3.filter(function(ele, idx, arr){})遍历数组元素，传入function()将返回值为true的元素组成一个新的数组对象返回

4.some(function(ele, idx, arr){})遍历数组元素，传入funciton()有一个元素返回值为true则整体返回true

5.every(function(ele, idx, arr){})遍历数组元素，传入funciton()有一个元素返回值为false则整体返回false

6.reduce(function(pre, cur, idx, arr){})从第一个元素开始遍历数组，累计值为返回值

7.reduceRight(function(pre, cur, idx, arr){}) 与 reduce()方法一致，不同于从最后一个元素开始

8.indexOf()返回元素在数组中第一次出现时的index

9.lastIndexOf()返回元素在数组中最后一次出现时的index
