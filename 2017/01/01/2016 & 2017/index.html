2016 & 2017

发表于 2017-01-01   |   分类于 me

和自己聊聊 2016 的一些事。

2016

年初

车巴巴商城二期重构完成

技术栈：Django + jinjia2 + jq

谈点什么：第一次学会用 jq 封装插件，翻译了 jq 官网插件开发文档。虽然到今天，我再也没有开发过一个 jq 插件，也再也没有用过一次 jq。但是翻译还是蛮好玩的一件事。好像这次开发并没有真正学到什么，忙着业务忙着业务，仅此而已。

四月

路享科技面试

技术栈：meteor + react

谈点什么：如果说上半年，什么时间最开心，那我想是清明节这三天了。

四月开始骚动着跳槽相关的事情，遇见李河山（路享科技 CEO & 前亚马逊高级工程师），给出了一道面试题 - 用 meteor + react 完成一款类微信的简易 IM 工具。

和山哥两个人忙乎了清明三天假，数据库设计 & meteor 学习 & react 学习，能够在自己开发的 IM 工具上聊天。那感觉，不说了。写代码真的是件非常好玩的事情。

虽然最后面试还是挂了，但是感谢山哥 & 河山。

总得来说，我的工作舒适区好像一文不值。😶

五月

财迷网络

技术栈：nodejs + vue + socket.io

谈点什么：一段直到今天的自由的、野蛮的、迷惘的自我发展时间。身边没了山哥给我指导，在很长一段时间（直到今天），一直处于一种巨大的自我怀疑之中，繁多的业务与需求，面对的困难不再是如何解决，而是如何才是更优的解决，不知道这是幸运还是不幸运，我开始意识到我究竟有多少东西不会，有多少东西我只是会用但却丝毫不了解。

这期间学了好多东西，有设计方面的 sketch、有设计模式、有刷过题、有正则表达式、有 css，有的浅尝辄止了，有的一直在学一直在用。

而在这学习的过程中，很难量化到自己究竟学到了多少，仔细想想，潜移默化的影响远远多于了实实在在记住的知识点。

一段没法评价的时间，毕竟这不是已经渡过的一段时间，而是一段将长久挣扎于其中的一段时间。

十月

上海

技术栈： vue2 全家桶 + mocha + karma + chai

谈点什么：公司搬家到上海来啦（四月后跳槽到广州的一家创业公司，目前一直就职其中，负责移动端 wap 页面开发）。

因为新的股东注资，新的项目也全新启动啦，一直忙乎到今天。开始在实际项目中运用一些设计模式 & tdd测试 & CSS 揭秘中学到一些实战技巧。

但是关于技术这条路上我一直走的方式，关于个人情绪的一些事，关于技术的担忧，关于工作的疲惫，关于想学的东西，也开始为自己担忧了起来，身上的戾气越来越重，开始不爽很多很多东西，不知道自己怎么了。

我总在想，我总想立马得到，我总是得不到，我总在找借口，我总在问自己 - 你究竟学会了没有 ？

挣扎在一片挥之不去的自我怀疑的迷雾之中。这种感觉可真他妈的烦啊。

2017

从 16 年五月开始的一段时间，不知道是不是每位青年（或者每位 Front-End Engineer）都会经历的一段时间，活得不怎么没心没肺。想得太多，得到的太少。年末面试了些人，感觉很多人对于技术的担忧似乎并没有我这么严重，我不知道这样是好还是坏。

我想要好多东西，我不够努力去争取这些东西，我一直在埋怨，我一直在累积身上的戾气。

愿 2017 年我能找到问题，毕竟我一定能求解。

唠唠叨叨半天，干脆写一下 17 年我一定要做的一些事。

  1. 一刷 JavaScript数据结构算法

  2. 一刷 高性能 JavaScript

  3. 一刷 函数式编程思想

  4. 一刷 你不知道 JavaScript 上册

  5. 二刷 CSS 揭秘

  6. 二刷 JavaScript 设计模式

  7. 用 React 编写一套组件库

  8. 用 vue2 编写一套组件库

  9. 参加三场或以上技术会议
